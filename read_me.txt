1.First need to install XAMPP server in the Pc. Download link(https://www.apachefriends.org/download.html).
2.Copy the cms folder to the htdocs folder of xampp.
3.In browser type(localhost/phpmyadmin) and create a table name = 'content' in the database
4.Import the 'content' file  to the database.
5.In browser type (localhost/cms) to open the project.
6.Click on the Registration to register as admin.
7.Click any post and then comment anything to show the output in the sidebar.
